MISSTURER

ภาพรวม (Overview)

การออกแบบ UI/UX ของเกม Missturer

การออกแบบ UI/UX ของเกม Missturer ที่ผมวางแผนโดยเน้นไปที่ความสะดวกในการใช้งาน UI การใช้สีที่มี Contrast ชัดเจน Project นี้ผมได้ออกแบบทุกอย่าง Art Style, UI/UX, Graphic Design

Role UI/UX Designer
Software Figma / Photoshop
Team Size 1 คน

Wireframe

เริ่มจากการวางแผนโครงสร้างของเกมว่าควรมีหน้าอะไรบ้างและทำงานเชื่อมต่อกันอย่างไร หลังจากนั้นจึงจัด Layout อย่างง่ายโดยใช้หลักการ Visual Hierarchy เพื่อจัดวางองค์ประกอบต่างๆ และใช้ 4-Grid System ในการทำระยะห่างระหว่าง UI

User Flow

ออกแบบ User Flow เพื่อจำลองประสบการณ์ผู้เล่นในเกมว่าจะพบเจออะไรบ้าง เราจะออกแบบประสบการณ์เพื่อให้ผู้เล่นสามารถเข้าใจและเล่นเกมได้อย่างง่ายดาย หลังจากออกแบบ User Flow แล้วจึงออกแบบ UI ให้เหมาะสมกับภาพและ Flow ที่เราวางแผนเพื่อสร้างประสบการณ์ที่ดี
