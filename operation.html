Operation Eclipse

ภาพรวม (Overview)

เรื่องราวของร่างโคลนที่ตื่นขึ้นมาในองค์กรลับ

ผู้เล่นได้รับบทบาทเป็นร่างโคลนที่หลุดออกจากแคปซูลภายในองค์กร Eclipse คุณต้องหาความจริงเกี่ยวกับตนเองและองค์กร Eclipse เกมนี้ผู้เล่นจำเป็นต้องใช้อาวุธที่หลากหลายในการต่อสู้ ทำให้เกิดคอมโบสุดเท่ รวดเร็วและลื่นไหล

Genre 2D Platformer Shooter
Role Programmer / Graphic Design
Software Unity / Figma / Capcut / AfterEffect
Team Size 3 คน

ฟีเจอร์สำคัญ (Key Features)

Combat System

IN GAME COMBAT

Gameplay

IN GAME PLATFORMER

ระบบต่อสู้ที่รวดเร็วและแปลกใหม่

ผู้เล่นสามารถเลือกอาวุธในการต่อสู้เพื่อได้รับสกิลพิเศษในการต่อสู้ เช่น Dash, Double Jump, Barrier ทำให้เกิดการสลับอาวุธเพื่อใช้ความสามารถพิเศษของตัวละคร และถ้าผู้เล่นสามารถทำคอมโบได้ ตัวละครของคุณจะได้บัฟพิเศษแบบสุ่มมากมาย

การเล่าเรื่องผ่านฉากในเกม

ด่านในเกมถูกออกแบบเพื่อให้ผู้เล่นได้สำรวจองค์กร "Eclipse" ที่เต็มไปด้วยความปริศนา เราได้ใช้หลักการ Level Design เพื่อให้ผู้เล่นดื่มด่ำกับ Gameplay และเนื้อเรื่องอย่างสมบูรณ์
