ภาพรวม (Overview)

ผมได้ตัดต่อคอนเทนต์เกี่ยวกับเกม Pokemon TCG โดยวิธีตัดต่อผมพยายามทำMotion GraphicและTypography
เพื่อนำเสนอเกมในรูปแบบที่ให้ผู้ดูสามารถเข้าใจง่าย

หมวดหมู่เกม (Genre)
Action / Soul-like
ตำแหน่ง (Role)
Game Developer / Graphic Design
ซอฟต์แวร์ (Software)
Unreal Engine / Photoshop / Github / Docs
สมาชิกทีม (Team Size)
3 คน

จุดเด่นเกม (Key Features)

AI Behavior Tree

เกมนี้มีระบบ AI ที่ใช้ Behavior Tree เพื่อควบคุมพฤติกรรมของศัตรูAI ในเกมนี้มี 3 ประเภทได้แก่
1.AI ตีใกล้ ทำหน้าที่วิ่งเข้าหาผู้เล่นแล้วโจมตีเมื่อHPเหลือครึ่งนึงจะบ้าคลั่ง
2.AI ตีกลาง ทำหน้าที่ยิงปืนไปยังผู้เล่นระหว่างนั้นจะรักษาระยะห่างเมื่อผู้เล่นเดินเข้าใกล้เกินไป
3.AI ตีไกล จะยืนโจมตีจากระยะไกล

Art Style

นำเสนอภาพในสไตล์ Cel Shade มีการสร้าง post process volume ขึ้นมาใหม่เพื่อให้มีความเหมาะสมกับงานภาพของเกม
รวมถึงการ Render outline ของวัตถุต่างๆภายในเกมและมีการใช้Level Design , Naigara VFX
